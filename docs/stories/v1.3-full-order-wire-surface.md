# v1.3: full order wire surface

## Summary

v1.3 is the follow-on to the grounded v1.2 order work. It closes the
remaining rare or lightly-evidenced order branches that were
intentionally deferred so v1.2 could land on solid protocol evidence.

## Target Scope

- Delta-neutral order extensions in `PlaceOrder` and `OpenOrder`.
- Scale order extensions in `PlaceOrder` and `OpenOrder`.
- Remaining rare `OpenOrder` post-status branches, including the
  specialized peg-style fields that still sit outside the grounded path.
- Full `CompletedOrder` detail extraction using the same grounded/shared
  order decoder sequence.
- Additional live capture scenarios for rare order branches before
  removing the remaining safe partial fallback.

## Requirements

- Every new branch must be justified by live captures or equivalent
  ground truth, not by guessed offsets.
- `OpenOrder` should only lose its remaining partial fallback once the
  deferred branches are capture-grounded and regression-frozen.
- `CompletedOrder` widening must land with capture-backed tests or
  transcript freezes for the expanded detail set.

## Acceptance

- Advanced rare-order captures are checked in and covered by codec tests.
- No remaining guessed offset logic in the deferred order branches.
- Public docs and message coverage no longer describe the deferred
  sections as unsupported once they are landed.
